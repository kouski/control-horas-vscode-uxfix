# Control de Horas Extra

Proyecto React + Vite listo para ejecutar en local.

## Cómo arrancarlo

```bash
npm install
npm run dev
```

Después abre en el navegador la URL que te muestre la terminal, normalmente:

```bash
http://127.0.0.1:5173/
```

## Cambios de esta versión

- horas directas en la celda del día
- cálculo automático al escribir
- doble clic en el día solo para observaciones y festivo manual
- ciudades solo en la celda mensual de ciudades
- espaciado y altura de celdas revisados para que los números se vean completos
